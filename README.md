# STEPHER HAIR - Premium Barbershop Website

A modern, responsive website for a luxury male grooming and barbershop business.

## 🌟 Features

- **Fully Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Single Page Application** - All content on one page with smooth scrolling navigation
- **Portfolio Gallery** - Separate page showcasing work with filtering system
- **Booking System** - Integrated appointment booking form
- **Contact Form** - Easy way for clients to get in touch
- **WhatsApp Integration** - Floating WhatsApp button for instant messaging
- **Animated Elements** - Smooth animations using AOS (Animate On Scroll)
- **Modern UI/UX** - Clean, professional design with gold accent colors

## 📁 Project Structure

```
STEPHER HAIR/
├── index.html          # Main homepage with all sections
├── portfolio.html      # Portfolio gallery with filtering
├── css/
```

## 📝 Customization

To customize the website:

1. **Colors**: Edit CSS variables in `style.css`
2. **Content**: Update text in `index.html` and `portfolio.html`
3. **Images**: Replace image URLs with your own
4. **Contact Info**: Update phone numbers, email, and address
5. **Services/Pricing**: Modify service cards and prices
